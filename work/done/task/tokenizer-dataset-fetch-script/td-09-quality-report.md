# task TD-09: 生成质量报告

状态：done

依赖：TD-08

## 目标

生成可审阅的四语语料质量报告，支持发现语言混入、异常重复和过滤偏差。

## 输入

- 最终四语语料、manifest 和处理统计。
- 固定人工抽样 seed。

## 执行事项

- 统计每种语言的行数、字符数、长度分布和来源占比。
- 统计各过滤 reason code、精确去重和近似去重比例。
- 使用稳定内容 ID 固定抽取人工检查样本。
- 检查语言混入、乱码、HTML、模板文本和异常重复。
- 报告只写统计、ID 和检查结论，不直接写入可能敏感的大段原文。
- 记录质量异常、接受理由和需要调整的配置项。

## 产物

- `data/tokenizer/reports/tokenizer_corpus_mvp.md`
- 可选的本地敏感样本检查文件，不进入 Git。

## 验收

- 四种语言均有完整统计和人工检查结论。
- 报告数字与 manifest 一致。
- 固定输入下统计部分可复现。
- 所有异常都有结论或后续处理项。
