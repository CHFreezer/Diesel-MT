# task TD-16E: 验证重复训练能力等价并冻结候选

状态：pending

依赖：TD-16D

## 目标

对 TD-16D 选定配方做独立复跑，以能力和 time-to-quality 验收统计等价，并只按 dev 冻结唯一正式候选。

## 原子边界

本 task 不修改数据、配方或预算，不访问 test，不要求模型 hash、逐步 loss 或权重逐 bit 相同。

## 执行事项

- 按预注册次数独立复跑最终配方，记录输入身份、seed、环境、墙钟、累计 token 和最佳 checkpoint。
- 比较总体及20路 dev BLEU/chrF、loss、实体/数字忠实度、脚本、空输出、source-copy 和达到质量门槛的时间。
- 输入、配置与 checkpoint 文件完整性必须可验证；训练数值只要求落入预注册能力容差。
- 不挑表现最好的一次掩盖不稳定，也不为失败追加预算；通过后冻结唯一候选和 test 授权记录。

## 产物与验收

- 重复训练能力等价报告、time-to-quality 分布、唯一候选和正式 test 授权。
- 正式 test 消耗仍为 `0/1`。
