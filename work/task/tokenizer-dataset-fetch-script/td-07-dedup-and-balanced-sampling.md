# task TD-07: 实现去重与均衡抽样

状态：pending

依赖：TD-06

## 目标

对清洗文本执行确定性精确去重、近似去重和字符预算抽样，生成四语均衡样本。

## 输入

- TD-06 的清洗文本流和稳定来源标识。
- profile 字符预算、seed 和 MinHash 配置。

## 执行事项

- 定义文本去重键和稳定内容 ID，使用明确的加密哈希算法。
- 对四语执行精确去重；对英文和中文额外执行 MinHash 近似去重。
- 固定 MinHash 分词单位、n-gram、哈希函数、阈值、seed 和并列项规则。
- 禁止使用 Python 内置 `hash()` 或依赖集合迭代顺序。
- 按稳定逻辑顺序和字符预算选样，固定预算边界样本处理方式。
- 记录去重前后数量、字符量、重复率和抽样拒绝数。

## 产物

- 去重和均衡抽样模块。
- 算法版本及参数配置。
- 可复现统计和测试向量。

## 验收

- 不同进程和不同并发度产生相同保留 ID 序列。
- 四语输出不超过配置预算，边界行为有明确测试。
- 相同 seed 输出字节级一致，不同 seed 的行为可解释。
- 精确与近似去重统计可核对。
