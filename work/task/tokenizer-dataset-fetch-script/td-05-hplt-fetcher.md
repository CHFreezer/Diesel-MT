# task TD-05: 实现 HPLT 3.0 获取器

状态：pending

依赖：TD-03、TD-04

## 目标

按 source lock 稳定获取 HPLT 3.0 分片，支持缓存、校验、恢复和离线重建。

## 输入

- HPLT registry 和 source lock。
- CLI 运行参数与输出目录。

## 执行事项

- 实现 map 和 `.jsonl.zst` 分片读取适配器。
- 只消费 lock 中允许的 WDS 分片，达到 profile 字符预算后停止。
- 将下载完成顺序与逻辑处理顺序分离，合并严格按 lock 顺序执行。
- 实现超时、有限重试、断点续传和已完成分片跳过。
- 下载完成后计算 SHA-256；缓存复用前重新校验。
- lock、远端内容或缓存校验不一致时快速失败。
- 在 `--use-cache --offline` 下仅使用本地缓存完成读取。

## 产物

- HPLT 3.0 获取器与缓存状态实现。
- 下载和校验错误类型。
- fixture 驱动的适配器测试。

## 验收

- 中断后重启不会重复下载已校验分片。
- 冷缓存和热缓存返回相同的逻辑文档序列。
- 并发度变化不影响输出顺序。
- 缓存损坏、远端变化和离线缺失均有明确失败结果。
